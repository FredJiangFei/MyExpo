# Built-in URL Schemes
    mailto	
        Open mail app, eg: mailto: support@expo.io
    tel	
        Open phone app, eg: tel:+123456789
    sms	
        Open SMS app, eg: sms:+123456789
    https / http	
        Open web browser app, eg: https://expo.io
